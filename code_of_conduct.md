# Code of Conduct

## Our Standards
We expect contributors and collaborators to be respectful, constructive, and professional.

Examples of positive behavior:

- Be patient and considerate
- Give and receive feedback in good faith
- Focus on the code and the problem, not the person
- Help others learn and improve

Examples of unacceptable behavior:

- Harassment, discrimination, or personal attacks
- Abusive, threatening, or intimidating language
- Deliberate trolling or disruption
- Sharing private information without consent

## Enforcement
Project maintainers may remove or limit participation for behavior that violates this code of conduct.

## Reporting Issues
If you experience or witness behavior that violates this code of conduct, report it privately to the project maintainers with:

- What happened
- When it happened
- Who was involved, if known
- Any supporting context or evidence

## Scope
This code of conduct applies to project discussions, code reviews, issues, pull requests, and related collaboration spaces.
